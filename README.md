# Heart Disease Prediction 🫀

This project uses Machine Learning to predict the likelihood of a patient having heart disease based on clinical parameters such as age, sex, cholesterol levels, and heart rate.

## 📊 Dataset
The dataset used in this project is the **Heart Disease Dataset**. It contains key features used to determine the presence of heart disease in patients.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** * `pandas` & `numpy` (Data Manipulation)
    * `plotly` (Interactive Visualizations)
    * `scikit-learn` (Machine Learning Model)
    * `pickle` (Model Serialization)
